Welcome to SIN ✘ - Solar System Visualization

SIN ✘ is an interactive 3D visualization of our solar system. This project aims to provide an engaging and educational experience for users to explore the planets, their orbits, and learn about the cosmic bodies in our solar neighborhood.

Planets

Explore the planets in our solar system, each with detailed information and realistic textures.

  • Accurate representation of planet sizes and orbits
  • Interactive zoom and rotation controls
  • Detailed information about each planet
  • Realistic textures and asteroid belt visualization

Project Info

This project is designed for astronomy enthusiasts, students, and anyone interested in learning more about our solar system. It serves as an educational tool that combines scientific accuracy with interactive web technologies to create an immersive learning experience.

Purpose:

The code serves as an interactive viewer for asteroids, allowing users to explore various asteroids in a 3D environment. Users can click on asteroids to view detailed information about them.

Key Features:

  • 3D Rendering: Utilizes Three.js to create a 3D scene with asteroids and a starfield.
  • Asteroid Data: Contains an array of asteroid objects with properties like name, diameter, image, and description.
  • User Interaction: Users can click on asteroids to display detailed information, including an image and description.
  • Responsive Design: Adjusts the view when the window is resized.

Target Audience:

This application is likely aimed at educational purposes, astronomy enthusiasts, or anyone interested in learning about asteroids in an engaging way.

Detailed Functionality:

  • Initialization: Sets up the 3D scene, camera, renderer, and controls.
  • Asteroid Creation: Generates 3D models of asteroids based on predefined data and positions them in the scene.
  • Starfield Creation: Creates a background of stars to enhance the visual experience.
  • Event Handling: Listens for user clicks to determine if an asteroid has been selected and displays its details.
  • Responsive Design: Adjusts the camera and renderer when the browser window is resized.
  • Animation Loop: Continuously updates the scene for smooth rendering and user interaction.

Key Features:

  • Accurate representation of planet sizes and orbits
  • Interactive zoom and rotation controls
  • Detailed information about each planet
  • Realistic textures and asteroid belt visualization
  • Educational tool for astronomy enthusiasts and students

Technologies Used:

  • HTML/CSS: For structuring and styling the web page
  • JavaScript: For adding interactivity and handling user inputs
  • Three.js: A JavaScript library used to create and display animated 3D graphics in a web browser
  • WebGL: A JavaScript API for rendering interactive 3D graphics within any compatible web browser
